        Coleção Filmes, Diretores e Produtoras

        Introduction

        REST API para acessar uma coleção de filmes, diretores e produtoras. Poderá consultar, cadastrar, alterar e excluir um item na coleção. Pesquisar por atributos específicos, e obter a paginação.

        Operações

        Usamos verbos RESTful para todas as operações e métodos.

        Verbo Descrição
        GET Usado para selecionar um ou mais itens. Sucesso retorna status code 200 com os itens do resultado json no body.
        POST Usado para criar um novo item. Sucesso retorna status code 201 com o id json do novo item no body.
        PUT Usado para atualizar um item. Sucesso retorna status code 200 sem conteúdo body.
        DELETE Usado para excluir um item. Sucesso retorna status code 204 sem conteúdo no body.

        Status Codes

        As respostas serão com um dos seguintes status codes.

        Code Descrição
        200 Success. Requisição completa.
        201 Success. Novo recurso criado.
        204 Success. Sem conteúdo para retornar. Retorna somente o status code.
        400 Bad Request - A requisição não pode ser interpretada.
        500 Server error.
